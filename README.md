# Curium for Cuphead
Curium is an open source cheat for Cuphead game. Tested on Cuphead v1.0.
## Features
- hard invincibility
- full invincibility
- unlimited super meter
## How to use
1. Download release .exe file or compile from source code.
1. Launch Cuphead.
1. Run curiumcheat-cuphead.exe as administrator.
1. Select mode.
1. Play!
